# Pizza Sales Analysis (SQL & Power BI)

## 📌 Project Overview
This project analyzes a pizza restaurant's sales dataset to discover key insights regarding customer preferences, peak ordering times, and revenue performance. Using SQL, I developed complex queries to calculate business metrics, while Power BI was used to build an interactive dashboard for visualizing sales trends.

## 🛠️ Tech Stack Used
* **Database/Querying:** MySQL / SQL
* **Data Visualization:** Power BI
* **Documentation:** GitHub Markdown

## 🔍 Key Data Insights Extracted (SQL)
* **KPIs:** Calculated Total Revenue, Average Order Value, Total Pizzas Sold, and Average Pizzas Per Order.
* **Sales Trends:** Identified peak hours and days (e.g., weekends) with maximum order volume to optimize staffing.
* **Product Performance:** Analyzed the top-selling and least-selling pizzas by revenue and quantity to improve menu management.

## 💡 Key Business Insights (Quick Review)
* **Peak Days & Hours:** Highest order volumes are recorded on Fridays and Saturdays, particularly between 5:00 PM and 8:00 PM.
* **Best Seller:** The Classic Category pizzas contribute the most to the total quantities sold and overall orders.
* **Revenue Driver:** The Supreme Category pizzas generate the highest revenue due to higher average item prices.
* **Size Preference:** Large-sized pizzas are the most popular choice among customers, driving maximum sales volume.
